Funcionamento e Atributos de Áudio e Vídeo

As tags <audio> e <video> funcionam como contêineres multimídia. Para garantir maior compatibilidade entre navegadores, adicionamos sub-tags <source> internas apontando para diferentes formatos (como MP3, OGG, MP4 e WebM).

Dicionário de Atributos Oficiais

Veja abaixo quais são e para que servem os principais atributos modificadores dessas tags:

Atributo Alvo Finalidade / Comportamento
src Ambos Define o caminho/URL do arquivo de mídia que deve ser carregado.
controls Ambos Instrui o navegador a renderizar os botões nativos de Play, Pause, barra de volume e progresso. Sem ele, a mídia fica invisível ou inacessível ao usuário padrão.
autoplay Ambos Força o início automático da mídia assim que a página é carregada (Geralmente requer o atributo muted associado devido a políticas anti-ruído dos navegadores).
loop Ambos Faz com que a reprodução retorne ao segundo zero imediatamente após sua conclusão, criando um ciclo infinito.
muted Ambos Inicia o reprodutor de mídia em modo silencioso por padrão.
preload Ambos Ajuda na otimização ao sugerir como a mídia deve pré-carregar (valores: none para não gastar dados, metadata para baixar o tamanho/duração, auto para baixar tudo).
poster Vídeo Define o endereço de uma imagem para funcionar como "capa de exibição" antes do usuário clicar no play.
width / height Vídeo Configura as dimensões em pixels da caixa do reprodutor diretamente no HTML.
